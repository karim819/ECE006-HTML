c++ language

C++ is a language of middle level allowing it the benefit of low-level programming (drivers, kernels)
And higher-level software (games, GUIs, desktop software etc.). All C and C++ have the same basic syntax and code structure

Applications of C++

  1. Application Software Development

    2. C++ programming was used to build almost all of the main operating systems such as Windows,
    Mac OSX and Linux. In addition to the operating systems, other browsers, such as Mozilla
    Firefox and Chrome, have written the central component using C++. C++ has also been
    used to build MySQL, the most popular database system

  2. Programming Languages Development

    3. C++ was commonly used in the development of modern programming languages such as C #, Java ,
    JavaScript, Perl, UNIX 's C Shell, PHP and Python, and Verilog etc.

  3. Games Development

    4. C++ is extremely fast, enabling programmers to do procedural programming for CPU-intensive functions,
    and having greater hardware power, as a result of which it has been widely used in gaming engine development.

  4. Embedded System

    5. C++ is used extensively in the production of medical and engineering applications such as MRI machine software, CAD / CAM high-end systems etc.

comparison between c and c++ :

c language c++ language
C uses functions for input/output. For example scanf and printf. C++ uses objects for input output. For example cin and cout.
C does not support reference variables. C++ supports reference variables.
C does not have namespace feature. C++ uses NAMESPACE which avoid name collisions.
C does not allow functions to be defined inside structures. In C++, functions can be used inside a structure.
