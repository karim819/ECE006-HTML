python language

Python is a common language used for programming. Guido van Rossum produced it,
and it was published in 1991. It has a basic syntax resembling English.
You will execute code as soon as you write it. This means prototyping can be very easy. Has a syntax
that helps developers to write less-line programs than some other programming languages.

it is used for :
  • web developmen
  • software development
  • mathematics
  • system scripting

Characteristics of Python

  • It supports both functional and formal methods of programming, and OOP.
  • This can be used as a scripting language or compiled into byte-codes to construct broad applications
  • It provides very high structured data types and supports dynamic form checks
  • It supports automatic garbage collection.
  • It can be easily integrated with C, C++, COM, ActiveX, CORBA, and Java.
