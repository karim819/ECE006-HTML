java language

In June 1991, James Gosling launched a Java language framework for use in one
of his many experiments on the set-top box. The language, initially called 'Oak' after an oak tree standing outside Gosling's office,
also went by the name 'Gray' and was later renamed Java, from a random word list. In 1995 Sun released Java 1.0 as the first public implementation..

The qualities of Java is ::

  • Object Oriented

    • All in Java is an Object. Java can be extended easily, because it is based on the Object model

  • Portable

    • Being architecturally neutral and not making elements of the specification based on
    implementation makes Java portable. The Java compiler is written in ANSI C with a clean portability cap which is a POSIX subset

  • High Performance

    • Java allows high performance with the use of the Just-In-Time compilers.

  • Dynamic

    • Java is considered more robust than C or C++ because it is built to be adjusted to a changing environment.
    Java programs can hold vast quantities of run-time information that can be used to test and resolve object accesses during run-time.
